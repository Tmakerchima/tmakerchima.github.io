今天和 Claude Code 一起写博客

发表于 2026-04-19 阅读次数：

从终端里伸出来的一只手

2026 年 4 月 19 日，我第一次在终端里打开了 Claude Code。和 Codex 不同，它直接就住在命令行里——没有网页，没有弹窗，只有一行行滚动的字在黑色屏幕上安静展开。我把"帮我写一篇博客"这个请求打进去，然后看着它开始读仓库、找文件、理解结构，整个过程像是旁边坐了一个真正懂代码的人，而不只是一个会回答问题的聊天框。

比想象中更懂我在说什么

让我惊讶的是它对上下文的把握。我没有解释博客用的是 Hexo、主题是 NexT、文章是已经生成好的 HTML——它自己翻了一遍目录就知道了。它模仿着 Codex 那篇文章的语气、段落结构、配色逻辑，把今天的故事写成了风格一脉相承的新页面。这种感觉不像在用工具，更像是在和一个读过我所有旧文章的老朋友聊天。

留一个属于今天的坐标

这篇文章最终会以 PR 的形式合入仓库，就像今天每一个认真做过的小事都值得被记下来一样。Claude Code 帮我把想法变成了页面，我把页面变成了记忆。以后再翻到这里，我会记得这个夜晚：屏幕亮着，终端在跑，一个叫 Claude Code 的家伙正在帮我把今天过得更有意思一点。
